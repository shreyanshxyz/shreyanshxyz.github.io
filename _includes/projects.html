CTracker - India's Covid Tracker

  • React.js, axios, Firebase

The web application has two parts. The first part displays the Covid-19 cases in the whole country by default, with an intuitive graph which can toggle between the confirmed, recovered and deceased cases. The second displays the statewise cases in the country with a bar graph corresponding to the respective state. The dropdown menu allows the user to choose any state of their choice and then see the cases of the respective state.
